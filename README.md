# Azure Active Directory Hybrid Lab

Author: John (Oladele) Adeboyejo  
Location: Glasgow, United Kingdom

## Project Overview

This project demonstrates a hands-on enterprise-style IT infrastructure lab combining:

- Windows Server Active Directory
- Microsoft Entra ID (Azure AD)
- Azure Virtual Machines
- Group Policy security controls
- Identity and access management
- Helpdesk troubleshooting simulations
- Azure cost optimization

The goal of this lab is to simulate real-world IT support and cloud administration tasks.
